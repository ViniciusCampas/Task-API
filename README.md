# Task API
Projeto de uma API Rest que Priorização de tarefas por análise de palavras-chave no título e descrição.

## 🚀 Tecnologias
- flask 
- spacy
- sqlite3

## 📌 Funcionalidades
- Criar tarefa
- Listar tarefas
- Atualizar tarefa
- Deletar tarefa

## ▶️ Como rodar
1. Clonar o repositório
2. Rodar o projeto
3. Acessar via Swagger/Postman
